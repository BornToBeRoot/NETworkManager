Binaries are now signed with a code signing certificate

May 24, 2024 Â· One min read
BornToBeRoot
Maintainer of NETworkManager

Starting with NETworkManager version 2024.5.24.0, the binaries and the installer are now signed with a code signing certificate.

The binaries and installer are built on AppVeyor directly from the GitHub repository. Build artifacts are automatically sent to SignPath.io via webhook, where they are signed after manual approval by the maintainer. The signed binaries are then uploaded to the GitHub releases page.

Special thanks goes to SignPath.io for providing free code signing and the SignPath Foundation for the free code signing certificate.
